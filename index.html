Projects

ATM Project

This project is based on the working of an Automated Teller Machine (ATM). The main objective of this project is to understand how basic banking transactions are performed using an ATM system. It allows users to securely log in using a PIN and perform operations such as cash withdrawal, balance inquiry, and mini statement. This project helps in learning concepts of authentication, transaction processing, and user interaction in a banking system.
